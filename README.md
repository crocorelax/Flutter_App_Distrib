
# Vending Vote App

A Flutter application developed for a business plan and coursework. It allows users to vote for their favorite products to influence the selection in local vending machines.

## Key Features
* **Product Voting:** Real-time feedback on machine inventory.
* **Business Integration:** Designed as a tool for vending operators to optimize supply.
* **Cross-Platform:** Built with Flutter for seamless iOS and Android deployment.

## Project Context
This project was created to bridge the gap between consumer preferences and automated retail distribution, focusing on market-driven inventory management.
